# v0.3.1 — Reindex Reliability

Reindexing now follows Unity's GUID identity model and handles asset changes
without silently damaging graph accuracy.

## Highlights

- Moves and renames preserve GUID identity instead of being treated as unrelated files.
- Asset and `.meta` changes both refresh the affected logical asset.
- Addressables data stays aligned when group assets change, move, or disappear.

## Reliability Fixes

- Duplicate GUIDs stop publication instead of producing an ambiguous graph.
- Same-path GUID replacements are reported and preserve unresolved old references.
- Reindexing builds a temporary database and replaces the last good index only after success.

## Usage

- Use normal `index_project` for fast incremental refreshes.
- Use `force: true` when guaranteed freshness is required.
